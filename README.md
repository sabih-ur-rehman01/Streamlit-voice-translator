# Streamlit-voice-translator

A simple and interactive Voice Translator Chatbot built using Python and Streamlit.  
This application allows users to:

-  Speak using a microphone
-  Enter text manually
-  Translate text into multiple languages
-  Listen to translated speech output

---

## Features

- Voice-to-Text using Speech Recognition
- Text Translation using Google Translator
- Text-to-Speech Audio Output
- Multiple Language Support
- Interactive Streamlit User Interface

---

## Technologies Used

- Python
- Streamlit
- SpeechRecognition
- gTTS (Google Text-to-Speech)
- deep-translator
